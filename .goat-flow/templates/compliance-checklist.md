# Template: Compliance Checklist

> Reference template. Installed to `.goat-flow/templates/` during setup.

## Compliance Checks

Compliance checks are opt-in. Only activate when the user explicitly mentions
HIPAA, GDPR, SOC2, PCI-DSS, or any other regulatory compliance requirement.

Run the standard security assessment first, then layer regulation-specific checks on top.

For each requested regulation, present gaps with this structure:

- Non-compliant
- Partially compliant
- Not assessed

Where possible, include a direct citation to the relevant clause or requirement
language.
